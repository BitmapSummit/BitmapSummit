CORE CONCEPT

What Is Bitmap?

Bitmap is an open-source consensus standard built on Bitcoin's Ordinals protocol. Proposed by Bitoshi Blockamoto on June 13, 2023, it allows anyone to claim ownership of a Bitcoin block by inscribing {block-height}.bitmap onto a satoshi.

Each Bitcoin block becomes a District — a parcel of digital real estate in the metaverse. Each transaction within that block maps to a Parcel of land, making every Bitmap unique based on real blockchain data.

"Some inventions are more like discoveries." — Bitoshi Blockamoto
// Bitmap Protocol — Open Source Standard
CLAIM "{block_height}.bitmap"
// inscribed onto a satoshi via Ordinals

BLOCK → District // 1 Bitcoin Block = 1 Metaverse District
TX → Parcel // Each transaction = 1 Parcel of land
VBYTES → Depth // Determines land depth in 3D
VALUE → Area // Block value determines land area

// Supply: infinite (new block = new Bitmap)
GENESIS: 0.bitmap // Block 0 — Satoshi's block
STANDARD: gitbook.bitmap.land
