Supply & Distribution

Bitmap has a unique supply model: theoretically infinite, but practically constrained. Every time a new Bitcoin block is mined, a new Bitmap becomes claimable. However, historical blocks (all blocks mined before the Bitmap protocol) were available at launch.

ON-CHAIN DATA SOURCES

Where to Find Live Data

🔍
Ordinals.com
The primary inscription explorer. Search any inscription number or content hash to verify Bitmap ownership. Shows inscription timestamp, sat number, and current owner address.
📈
NFT Price Floor
nftpricefloor.com/bitmap tracks floor prices, sales volume, holder counts, and market cap over time. Essential for tracking Bitmap's market performance.
🗺️
bitmap.land / atlas.land
The official Bitmap ecosystem portal. View districts, explore block data visualizations, and access Atlas OS — an on-chain operating system built for Bitmap holders.
📡
Mempool.space
Track Bitcoin blocks in real time. See block sizes, transaction counts, fees, and timing — all the raw data that determines each new Bitmap's properties.
🔬
Unisat Explorer
Unisat's inscription explorer provides Bitmap-specific search and filtering. See which blocks are claimed, available, and their inscription IDs.
📊
BRC-420 Analytics
brc420.io and rcsv.io provide protocol-level analytics for BRC-420 assets built on Bitmap — royalties, volume, active projects, and market data.
